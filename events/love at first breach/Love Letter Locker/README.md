## Steps to access the challenge

1. Turn on vpn. If you don't have any, follow the steps in the TryHackme's [OpenVPN](https://tryhackme.com/room/openvpn) room
2. Press the "Start Machine" button in the Target machine box to activate the target machine ip.

3. The MACHINE_IP is now replaced with target's IP
4. Copy the link
5. Paste the link in your preferred browser
6. You're in.
